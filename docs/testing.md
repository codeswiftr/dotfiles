## üõ†Ô∏è **Testing Tools Overview**

### **üåê Bruno - API Testing**
Modern, Git-friendly API testing tool for FastAPI backends.

**Key Features:**
- ‚úÖ **Git-friendly** - Text-based collections, version control ready
- ‚úÖ **CLI support** - Scriptable and CI/CD friendly  
- ‚úÖ **No cloud dependency** - Works offline
- ‚úÖ **Modern UI** - Better UX than Postman
- ‚úÖ **Environment management** - Dev/staging/prod configs

**Quick Commands:**
```bash

# Run collection from CLI

# Switch environment

```

### **üé≠ Playwright - E2E Testing**
Modern web testing framework for Lit PWA applications.

**Key Features:**
- ‚úÖ **Multi-browser support** - Chromium, Firefox, Safari
- ‚úÖ **Auto-wait** - No flaky tests from timing issues
- ‚úÖ **TypeScript first** - Better DX for modern frontends
- ‚úÖ **Built-in debugging** - Screenshots, videos, traces
- ‚úÖ **Mobile testing** - PWA compatibility

**Quick Commands:**
```bash
# Run tests

# Run with UI mode

# Debug specific test

# Generate test code
pw-codegen localhost:3000
```

**Included Packages:**

- ‚úÖ **pytest-xdist** - Parallel execution
- ‚úÖ **pytest-benchmark** - Performance testing

**Quick Commands:**

## üö® **Troubleshooting**

### **Common Issues**

**Playwright browser installation:**
```bash
bunx playwright install
```

**Python dependency issues:**
```bash
uv sync  # Refresh Python environment
testing-init python  # Reinstall test dependencies
```

**k6 permission issues:**
```bash
chmod +x loadtest.js
```

**Swift testing setup:**
```bash
# Ensure Xcode Command Line Tools
xcode-select --install
```

### **Performance Issues**

**Slow test startup:**
```bash
# Use fast mode for testing
export DOTFILES_FAST_MODE=1
```

**Memory issues with large test suites:**
```bash
# Run tests in smaller batches
pt-parallel -n 2  # Limit parallel workers
```
